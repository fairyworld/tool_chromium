# Accessibility-Specific Pre-authorized Operations

In addition to the generic operations in the Hub's `shared_workflows.md`, the
following accessibility-specific commands are pre-authorized:

- **Discovery Scripts:** `python3 scripts/find_accessibility_issues.py` (and any
  arguments)
- **Code formatter:** `git cl format`
- **Unit tests:** `tools/autotest.py` (and any arguments)
- **Build checker:** `gn check` (and any arguments)
